# Amazon-Order-Sales-Performance-Dashboard
This project focuses on analyzing Amazon e-commerce order data to understand sales performance, order trends, customer locations, and fulfilment status using Power BI.  The dashboard provides interactive insights that help businesses track revenue, monitor order status, and identify high-performing categories and cities.

🎯 Objectives
* Track total sales, orders, and quantity
* Analyze category performance
* Monitor order fulfilment status
* Identify top shipping cities and states
* Understand sales trends over time

📊 Dashboard Features
* KPI Cards (Total Orders, Quantity, Average Order Value)
* Sales Trend Analysis (Year, Quarter, Month, Day)
* Category-wise Sales Visualization
* Order Status Distribution (Donut Chart)
* Geographic Shipping Analysis (Map)
* City-level Sales Comparison
* Interactive Filters

🛠 Tools & Technologies
* Power BI
* DAX
* Data Cleaning
* Data Modeling
* Visualization Design

💡 Key Insights
* Certain product categories contribute most revenue
* Major cities dominate shipping volume
* Order fulfilment status shows operational efficiency
* Seasonal trends impact sales

🚀 Future Improvements
* Customer segmentation
* Profit analysis
* Return prediction model
* Real-time dashboard integration
